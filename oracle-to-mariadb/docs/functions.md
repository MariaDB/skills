# Oracle-Compatible Functions

| Function | Available Since |
|---|---|
| `DECODE()` | 10.3 |
| `CHR()`, `SUBSTR()` with position 0 | 10.3 |
| `ADD_MONTHS()` | 10.6 |
| `TO_CHAR()` | 10.6 (FM padding-suppression format added in 12.0) |
| `SYS_GUID()` | 10.6 |
| `ROWNUM` | 10.6 |
| `TO_NUMBER()` | 12.2.1 |
| `TO_DATE()` | 12.3 (native; on older versions use `STR_TO_DATE()`) |
| `TRUNC()` | 12.2 |
